influx-cli
===============================

version number: 0.0.1
author: Stephen Chen

Overview
--------

An interactive command-line influxdb cli.

Installation / Usage
--------------------

To install use pip:

    $ pip install influx_cli


Or clone the repo:

    $ git clone https://github.com/RPing/influx_cli.git
    $ python setup.py install
